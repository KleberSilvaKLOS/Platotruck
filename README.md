# Título do Projeto

Platotruck

## Descrição

Este projeto é uma aplicação web que permite aos usuários gerenciar produtos e suas informações de forma eficiente.

## Instruções de Instalação

1. Clone o repositório:
   ```bash
   git clone <URL do repositório>
   ```
2. Navegue até o diretório do projeto:
   ```bash
   cd Platotruck
   ```
3. Abra o arquivo `index.html` em um navegador para visualizar a aplicação.

## Uso

Após abrir o arquivo `index.html`, você poderá interagir com a aplicação e gerenciar os produtos conforme necessário.

## Estilos e Temas

Este projeto utiliza um sistema de temas que permite alternar entre um tema claro e um tema escuro. As definições de estilo estão localizadas no arquivo `style.css`.

### Tema Claro (Padrão)

O tema claro é o padrão da aplicação e é definido nas variáveis CSS no arquivo `style.css`.

### Tema Escuro

Quando a classe `.dark` é adicionada ao elemento `body`, a aplicação muda para o tema escuro, proporcionando uma experiência visual diferente para os usuários.

As classes e estilos são gerenciados de forma a garantir uma transição suave entre os temas, melhorando a usabilidade e a estética da aplicação.
